# Frontend Design

## Problem

需要定义前端页面布局、视觉风格和交互细节，确保黑白简约风格的一致性和良好的用户体验。

### 视觉风格

- **主色调**：纯黑 `#000` + 纯白 `#FFF`
- **辅助灰**：`#1A1A1A`（侧边栏）、`#F5F5F5`（消息区）、`#888`（次要文字）
- **边框**：`1px solid #E0E0E0`
- **字体**：系统无衬线字体栈，思考过程用 `font-style: italic`
- **圆角**：消息气泡 8px，按钮 4px
- **无阴影、无渐变、无装饰图标**

### 消息气泡

- 最大宽度 80%，内容超长时换行（`break-words`）
- 流式消息固定宽度 `w-[45%]`（非 `max-w`，避免空内容时气泡缩窄）
- 行间距 `leading-normal`（约 1.5）
- 气泡左上角显示时间（`YYYY/M/D HH:mm:ss`），`text-text-muted text-[11px]`
- 助手消息使用 `marked` + `highlight.js` 渲染 Markdown
- 用户消息保持纯文本
- 代码块使用 highlight.js 语法高亮，深色背景（`#1a1a1a`）
- 点击代码块可复制代码内容
- 无编辑按钮和编辑功能（已移除）

### 复制按钮

- 气泡外右下角
- 默认隐藏，hover 消息时完全可见（opacity-100）并微放大（scale-105）
- 点击复制内容到剪贴板
- 1.5 秒后图标变绿色勾

### 思考中状态

- 输入框、复杂度选择器禁用
- 发送按钮变为旋转 loading 图标

### 打字机效果

- `content_delta` 事件写入缓冲区，不直接追加到显示内容
- `contentBuffer`：接收 SSE 的 `content_delta`，累积未显示内容
- `startTypewriter()`：启动 25ms 间隔定时器，每次从缓冲区取 3 字符追加到 `msg.content`
- `stopTypewriter()`：停止定时器并刷新剩余缓冲区内容
- `done` 事件时刷新缓冲区剩余内容，确保完整显示

### 流式期间切换会话

- `streamingConvId` 跟踪当前流式消息所属会话
- SSE 读取循环中检测到 `streamingConvId !== conversationId` 时终止流处理
- `finally` 块只在 `convStore.activeId === conversationId` 时执行 `fetchMessages`
- 切换会话时清空旧会话的 `streamingMessage` 和 `isStreaming` 状态

## Acceptance Criteria

- 消息气泡最大宽度 80%，流式消息固定 `w-[45%]`
- 思考过程默认折叠，步数按 action 数量计数
- 复制按钮 hover 显示，点击复制，1.5s 变绿色勾
- 删除对话弹出自定义确认弹窗
- 思考中时输入框禁用，发送按钮变 loading
- 打字机效果：缓冲区逐字输出，done 时刷新剩余
- 流式期间切换会话正确终止旧流
- 代码块语法高亮，点击可复制
- 气泡左上角显示时间

## Changes by File

### `client/src/components/ChatArea.vue`

主聊天区域：欢迎页/聊天内容切换、System Prompt 弹窗

### `client/src/components/ChatInput.vue`

输入框：发送、换行、禁用状态、自动扩展、loading

### `client/src/components/MessageBubble.vue`

消息气泡：思考折叠、分支导航、复制按钮、Markdown 渲染、时间显示

### `client/src/components/ConversationList.vue`

侧边栏：对话列表、删除确认弹窗

### `client/src/components/ThoughtStep.vue`

思考步骤渲染

### `client/src/components/BranchNavigator.vue`

分支切换导航

### `client/src/assets/main.css`

全局样式：`.markdown-body`、`.hljs`、blink 动画

### `client/src/stores/message.ts`

流式消息管理：SSE 事件处理、打字机效果、缓冲区

## What This Enables

- 黑白简约一致的视觉风格
- 实时流式交互体验
- 思考过程透明展示

## What This Drops

- 无阴影、无渐变、无装饰图标（极简风格取舍）
